### 训练并保存模型
```python run.py --alg=ppo2 --policy_type=CustomPolicy --num_timesteps=1e6 --log_path=./log_test/ --save_path=./save_test/1e6.zip```  

### 观察训练模型效果
```python run.py --play --load_path=./save_test/1e6.zip```
